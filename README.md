# sorting-visualizer
This project is a high-performance, real-time sorting visualizer designed to bridge the gap between low-level algorithmic efficiency and modern interactive UI frameworks. Unlike standard visualizers built entirely in JavaScript, this tool utilizes a native C++ engine to handle the heavy computational logic of sorting algorithms. Using Pybind11, this high-speed backend is bound to a Python-based Streamlit frontend. The result is an educational tool that not only visualizes core data structures and algorithms in action but also demonstrates how to successfully integrate C++ performance with Python's rapid UI prototyping capabilities.
